  - Lists PyInstaller as build dependency
  - No runtime dependencies (uses standard library)

  - Auto-generated on first run
  - Stores paths to EZCAD2 and LightBurn drivers
  - JSON format for easy editing
  - Validated on load

WORKFLOW
================================================================================

DEVELOPMENT WORKFLOW:
1. Edit GalvoSwap.py
2. Test with: python GalvoSwap.py
3. Build with: pyinstaller --onefile --noconsole --name=GalvoSwap GalvoSwap.py
4. Test dist/GalvoSwap.exe
5. Distribute GalvoSwap.exe

END USER WORKFLOW:
1. Download GalvoSwap.exe
2. Run as Administrator
3. Complete setup wizard
4. Use to switch drivers


IMPORTANT NOTES
================================================================================

✓ GalvoSwap.exe is completely standalone
✓ No Python installation required for end users
✓ No external dependencies needed
✓ Configuration file created automatically
✓ Must run with Administrator privileges
✓ Works on Windows 7, 8, 10, 11
